# Thinking

Unified skill for structured thinking and idea development.

## Workflow Routing

| Request Pattern | Route To |
|---|---|
| first principles, decompose, challenge assumptions | Workflows/FirstPrinciples.md |
| iterative depth, multi-angle analysis | Workflows/IterativeDepth.md |
| brainstorm, creative exploration | Workflows/BeCreative.md |
| debate, weigh perspectives | Workflows/Council.md |
| critique, red-team, stress test | Workflows/RedTeam.md |
| hypothesis, experiment, improve | Workflows/Science.md |

## Thinking Rules

- Make assumptions explicit.
- Separate facts, hypotheses, and decisions.
- Keep reasoning traceable and concise.
