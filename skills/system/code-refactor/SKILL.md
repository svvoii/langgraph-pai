# CodeRefactor

Structured code refactoring skill with explicit safety and validation behavior.

## Workflow Routing

| Request Pattern | Route To |
|---|---|
| Refactor, cleanup, improve code structure | Workflows/Refactor.md |

## Safety Rules

- Read before write and preserve behavior.
- Keep changes scoped to requested files and features.
- Prefer incremental updates over broad rewrites.
- Validate with project checks and tests before completion.
